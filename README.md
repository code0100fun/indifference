# Indifference

For when you just need `with_indifferent_access`. Stolen 100% from ActiveSupport.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'indifference'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install indifference
